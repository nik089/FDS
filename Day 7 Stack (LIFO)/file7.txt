3Ô∏è‚É£ Stack Operations (Only 3 You Must Know)
| Operation      | Meaning              |
| -------------- | -------------------- |
| **Push**       | Add item to top      |
| **Pop**        | Remove item from top |
| **Peek / Top** | See top item         |

5Ô∏è‚É£ Stack in JavaScript (Very Basic)

        let stack = [];
        // Push
        stack.push(1);
        stack.push(2);
        stack.push(3);
        // Pop
        stack.pop(); // removes 3
        console.log(stack); // [1, 2]

6Ô∏è‚É£ Why Do We Use Stack?
    Because stack is perfect when:
    You need to go back
    You need undo
    You need reverse
    You need last action first

=======================================================
7Ô∏è‚É£ When Is Stack Used? (IMPORTANT)
        | Use Case              | Why Stack                |
    | --------------------- | ------------------------ |
    | Undo / Redo           | Last action undone first |
    | Browser back button   | Last page visited        |
    | Function calls        | Call stack               |
    | Expression evaluation | Operator precedence      |
    | Parenthesis checking  | Nested structure         |
    | Reverse string        | LIFO order               |
